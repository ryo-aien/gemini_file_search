# Sample Files

This directory contains sample documents for testing the Gemini File Search API.

## Files

1. **sample1.txt** - Plain text overview of File Search features
2. **sample2.md** - Markdown documentation about getting started
3. **sample3.json** - JSON data about API pricing and tiers

## Usage

These files can be used to:
- Test file upload functionality
- Verify document processing
- Test search queries
- Demonstrate different file format support

## Example Queries

After uploading these samples to a File Search Store, try these queries:

- "What are the key features of File Search?"
- "How much does the Pro tier cost?"
- "What file types are supported?"
- "What is the default chunk size?"
- "Tell me about RAG systems"
